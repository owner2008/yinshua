# 不干胶印刷报价系统

本仓库按 `开发落地执行文档.md` 开始落地，当前优先实现第一阶段的报价闭环。

## 当前结构

```text
apps/
  api/      NestJS 后端，优先实现报价引擎
  admin/    后台管理端占位
  client/   微信小程序 / H5 用户端占位
database/   数据库结构与种子数据
docs/       API、执行计划、验收文档
```

## 开发环境

本项目已内置便携 Node 开发环境：

```text
.tools/node
```

版本：

- Node.js：22.22.1
- npm：10.9.4
- pnpm：9.15.9

PowerShell 进入开发环境：

```powershell
.\scripts\dev-env.ps1
```

CMD 进入开发环境：

```cmd
scripts\dev-env.cmd
```

如果当前终端没有加载环境脚本，可在命令前临时加入：

```powershell
$env:Path=(Resolve-Path '.tools\node').Path + ';' + $env:Path
```

后端启动：

```powershell
pnpm --dir apps/api start
```

数据库初始化：

```powershell
copy apps\api\.env.example apps\api\.env
# 修改 apps\api\.env 中的 DATABASE_URL
pnpm --dir apps/api prisma:push
pnpm --dir apps/api db:seed
```

后端地址：

```text
http://127.0.0.1:3000/api
```
