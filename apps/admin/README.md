# 后台管理端

后台管理端按第二优先级落地，服务于报价引擎配置。

## 第一阶段页面

1. 产品分类管理。
2. 产品管理。
3. 报价模板管理。
4. 材料管理。
5. 材料价格管理。
6. 工艺管理。
7. 工艺价格管理。
8. 印刷价格管理。
9. 报价规则管理。
10. 报价单管理。
11. 报价快照查看。
12. 操作日志。
13. 基础库存。

## 技术建议

- React + TypeScript + Ant Design
- 或 Vue 3 + TypeScript + Element Plus

后台不需要和用户端使用同一套技术栈，优先保证开发效率、权限清晰和表单配置体验。
