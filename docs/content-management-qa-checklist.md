# 内容管理功能验收清单

本文用于验收以下 3 类内容管理能力是否完整可用：

- 企业介绍
- 首页头部配置（Logo + Banner）
- 分类设备展示

## 验收前准备

1. 启动本地环境：

```powershell
powershell -ExecutionPolicy Bypass -File scripts\dev-env.ps1
. .\scripts\start-mysql.ps1
pnpm --dir apps/api start:dev
pnpm --dir apps/admin dev
pnpm --dir apps/client dev
```

2. 打开管理端：

- 后台管理端：`http://127.0.0.1:5173`
- H5 用户端：`http://127.0.0.1:5174`
- API：`http://127.0.0.1:3000/api`

3. 使用管理员账号登录后台，进入“内容管理”菜单。

## 一、企业介绍

### 后台操作

1. 在“企业介绍”页填写：
- 标题
- 副标题
- 封面图
- 图集
- 正文
- 联系电话
- 微信号
- 地址

2. 点击“保存企业介绍”。

### 预期结果

- 保存成功后出现成功提示。
- 页面刷新后内容仍然存在。
- 封面图和图集图片可以正常预览。
- 状态为“启用”时，前台可见；状态为“停用”时，前台隐藏。

### 前台验收

1. 打开 H5 首页。
2. 检查企业介绍区是否展示：
- 标题与副标题
- 封面图
- 图集
- 正文
- 电话、微信、地址

3. 打开小程序首页。
4. 检查企业介绍区内容是否与后台一致。

## 二、首页头部配置

### Logo 与站点信息

1. 在“首页头部”页填写：
- 站点名称
- 站点副标题
- Logo 图片
- 头部文案

2. 点击“保存头部配置”。

### 预期结果

- 后台预览区同步展示最新 Logo 和文案。
- H5 顶部站点名称、Logo、头部文案更新。
- 小程序首页顶部 Logo、标题、副标题更新。

## 三、Banner 配置

### 后台操作

1. 新增 Banner，填写：
- 标题
- 副标题
- Banner 图片
- 移动端图片
- 跳转类型
- 跳转值
- 按钮文案
- 排序
- 状态
- 开始时间
- 结束时间

2. 保存后再编辑一次，修改标题或图片。

### 预期结果

- 列表页能显示 Banner 缩略图、标题、跳转信息、状态。
- 可按标题搜索、按状态筛选。
- 结束时间不能早于开始时间。
- 当跳转类型不是“无跳转”时，跳转值必须填写。

### 前台验收

1. H5 首页首屏展示第一个启用中的 Banner。
2. 其余 Banner 展示在 Banner 列表区。
3. 点击主按钮后跳转符合配置：
- 产品：跳产品详情
- 分类：跳产品列表并带分类
- 自定义：打开配置链接

4. 小程序首页首屏展示 Banner，点击后跳转行为正确。

## 四、分类设备展示

### 后台操作

1. 新增设备展示，填写：
- 所属分类
- 设备名称
- 展示标题
- 主图
- 图集
- 说明
- 规格参数 JSON
- 排序
- 状态

2. 再编辑一条已有记录，修改标题和参数。

### 预期结果

- 列表可显示主图、设备名、所属分类、图集数量、参数数量。
- 可按设备名称搜索、按所属分类筛选。
- JSON 非法时不能提交。
- 状态为“启用”时前台可见，停用时不可见。

### 前台验收

1. H5 首页展示分类设备区。
2. 每条设备展示包含：
- 主图
- 标题或设备名称
- 分类名称
- 说明
- 参数列表

3. 小程序首页展示同样内容。

## 五、图片上传

### 后台操作

1. 在企业介绍、Logo、Banner、设备展示中分别使用“上传图片”按钮上传。

### 预期结果

- 上传成功后返回 `/uploads/content/...` 地址。
- 后台表单自动填入该地址。
- 图片预览正常显示。
- 管理端可以复制图片地址。

### 接口验收

1. 访问上传后的图片 URL。

### 预期结果

- 返回 `200`
- 图片可直接打开

## 六、回归检查

1. H5 首页打开正常，无白屏。
2. 小程序首页打开正常，无报错。
3. 产品列表和产品详情页图片仍能正常显示。
4. 内容管理页打开正常，新增、编辑、筛选都可用。
5. 管理端构建通过：

```powershell
pnpm --dir apps/admin build
```

6. H5 构建通过：

```powershell
pnpm --dir apps/client build
```

## 七、验收结论

以下项目全部通过时，可认为这一轮内容管理功能完成：

- 后台可配置
- 图片可上传
- H5 可展示
- 小程序可展示
- 跳转行为正确
- 构建通过
